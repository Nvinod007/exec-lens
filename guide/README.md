## Who this is for

- **Learners** — see call stack, queues, scope, closures, hoisting, TDZ, and `this` while code runs.
- **Teachers** — use built-in examples or paste snippets; step through async behavior JSV9000-style tools miss.
- **Contributors** — agent/planning context lives in local `docs/` (gitignored); not required to use the app.

## Quick workflow

1. Pick an example or write custom code.
2. Click **Run** (`Mod+Enter`) — execution is recorded as a step timeline.
3. **Play** or step with `F7`/`F8`/`F9` (in editor) or arrows/`Space` (outside editor).
4. Watch the editor highlight, call stack, queues, scope, and console update together.
